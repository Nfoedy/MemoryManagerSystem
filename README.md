# Memory Manager System (C++)
Sistema di Gestione della Memoria sviluppato in C++, progettato per esplorare i concetti fondamentali della gesione manuale
della memoria utilizzati nei game engine e nelle applicazioni ad alte prestazioni.

Questo progetto è stato realizzato come parte di un percorso di studio in Advanced C++, con l'obiettivo di comprendere a fondo 
come la memoria viene allocata, tracciata ed ottimizzata.

# Obiettivi
Gli obiettivi principali del progetto sono:
- Comprendere il funzionamento della memoria dinamica in C++
- Costruire un sistema di gestione della memoria custom
- Tracciare allocazioni e rilevare memory leak
- Migliorare le performance tramite allocatori dedicati
- Simulare sistemi reali utilizzati nei game engine
